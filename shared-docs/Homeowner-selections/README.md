# Homeowner Selections

## What's Here

This folder holds color and material selection documents that homeowners sign off on before or during rehab projects.

### Files

| File | Type | Purpose |
|---|---|---|
| `Color Selection25.docx` | Word document | Blank or generic color selection form for 2025 projects |
| `Color Selection25- LC2025 Tampa Rehab.docx` | Word document | Lionize Construction version of the 2025 color selection form, specific to Tampa Rehab program projects |

Both files are `.docx` format. They could not be read as plain text — a Word-compatible application is needed to view their full content.

---

## What This Tells Us About the Current Process

Homeowners choose paint colors, finishes, or other material options through a paper/Word document process. The contractor (Luz) likely prints these forms, walks through selections with the homeowner, gets a signature, then files the signed copy.

There appear to be two versions: a generic one and a City of Tampa rehab-specific one — suggesting the form is being customized per program but still managed as individual Word files.

---

## Pain Points Observed

- Color/material selections stored only as Word files with no version control — easy to accidentally edit or overwrite
- No clear naming convention linking selections to a specific property address or contract number
- No digital signature workflow — selections likely require printing and wet signatures
- If a homeowner later disputes a selection, there's no audit trail beyond the file itself
- No record of when selections were made, revised, or approved
