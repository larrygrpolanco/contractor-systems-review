## What This Tells Us About the Current Process

Permits are applied for through the **Accela Citizen Access portal** (aca.tampagov.net). The screenshots suggest permits are being pulled manually — Luz navigates the portal step by step for each project, searching by address, uploading documents, and paying fees.

The Excel file acts as a personal cheat sheet: it stores URLs, fee estimator links, and job-specific notes that aren't documented anywhere else.

**Key contacts / numbers found in these docs:**
- ACA Portal help: 813-274-3100, option 1
- Land Development (address issues): 813-274-3100, option 2

## Pain Points Observed

- No central log of permit numbers, statuses, or expiration dates across jobs
- Screenshots saved as individual PDFs rather than organized by project — hard to reference later
- The Excel "instructions" file mixes procedural notes with project-specific data (roofing specs, vendor names, fee estimates) — fragile single point of reference
- Permit tracking is entirely manual; nothing alerts when a permit needs inspection or is about to expire
