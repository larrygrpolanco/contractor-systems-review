# Final Payment Closeout Checklist — TEMPLATE

> Copy this file for each project. It lists every document the City of Tampa
> requires to release **final payment** on a JOC housing-rehab job, who is
> responsible for each, and whether a missing/unsigned item will block payment.
>
> **How to use it three ways:**
> 1. **In Notion** — import `closeout-checklist-notion-import.csv` as a database,
>    then duplicate the rows per project and track Status on a board.
> 2. **On paper / in the repo** — fill the tables below directly.
> 3. **With Claude ("cowork")** — paste this filled-in checklist and ask
>    *"What's still outstanding and what's blocking payment?"* before submitting.
